# ChurrOS Brand Book

Bienvenido al Brand Book oficial de ChurrOS.

Este documento define la identidad visual del proyecto y establece las reglas para el uso del logotipo, colores, tipografía e identidad gráfica.

El objetivo es garantizar una imagen consistente en todos los componentes del proyecto.

---

# Identidad

ChurrOS es una distribución Linux moderna, abierta y desarrollada por la comunidad.

Nuestra identidad se basa en cinco pilares:

- Simplicidad
- Comunidad
- Modernidad
- Rendimiento
- Aprendizaje

---

# Filosofía

El diseño de ChurrOS debe transmitir:

- confianza
- claridad
- minimalismo
- tecnología
- comunidad

No buscamos un diseño corporativo.

Queremos que el sistema se sienta cercano y moderno.

Todos los recursos gráficos del proyecto deberán seguir esta guía.
