# ChurrOS UI

ChurrOS UI es la biblioteca oficial de componentes gráficos de ChurrOS.

Su objetivo es proporcionar una experiencia visual consistente en todas las aplicaciones oficiales del proyecto.

## Objetivos

- Componentes reutilizables.
- Diseño consistente.
- Alto rendimiento.
- Fácil mantenimiento.

## Aplicaciones que la utilizarán

- ChurrOS Welcome
- ChurrOS Settings
- ChurrOS Installer
- Futuras herramientas oficiales
