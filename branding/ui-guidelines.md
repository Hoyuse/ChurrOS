# ChurrOS UI Guidelines

## Filosofía

- Minimalista
- Moderna
- Rápida
- Consistente

## Bordes

12px

## Espaciado

- 8px
- 16px
- 24px
- 32px

## Sombras

Muy suaves.

## Animaciones

150–200 ms

Nunca exageradas.

## Transparencia

85–90%

No usar efectos excesivos.

# Paleta oficial

## Colores principales

- Naranja principal: `#F97316`
- Hover: `#FB923C`
- Fondo: `#1F1F1F`
- Paneles: `#2A2A2A`
- Texto: `#F5F5F5`
- Texto secundario: `#A8A8A8`
