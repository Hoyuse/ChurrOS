# ChurrOS Welcome

ChurrOS Welcome será la aplicación oficial de bienvenida de ChurrOS.

Su objetivo es ofrecer una excelente primera experiencia al usuario, permitiéndole conocer el sistema, acceder a la documentación y realizar las configuraciones iniciales de manera sencilla.

---

## Objetivos

- Dar la bienvenida al usuario.
- Mostrar información básica del sistema.
- Facilitar el acceso a la documentación.
- Acceder rápidamente al repositorio del proyecto.
- Centralizar las primeras acciones después de instalar ChurrOS.

---

## Funciones planificadas

### Primera versión

- Pantalla de bienvenida.
- Información del sistema.
- Acceso a la documentación.
- Acceso al repositorio de GitHub.
- Información de la versión instalada.

### Versiones futuras

- Instalación de aplicaciones recomendadas.
- Configuración del entorno de escritorio.
- Selección de temas.
- Selección de fondos de pantalla.
- Configuración para desarrollo.
- Configuración para gaming.

---

## Tecnologías

La primera versión será desarrollada utilizando:

- Python
- GTK4
- Libadwaita

---

## Filosofía

ChurrOS Welcome seguirá la filosofía del proyecto:

- Simple.
- Rápido.
- Elegante.
- Fácil de utilizar.
- Enfocado en la comunidad.
